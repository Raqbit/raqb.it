Hello!

My name is Ramon, I am currently 16 years old and I create software.

I started by writing server-side plugins for the game Minecraft in java, and later started experimenting with other languages. Now I'm focusing a lot on the quickly evolving web platform, playing a lot with javascript frameworks like Angular, new web standards like service-worker & web-components, and on the server side I like to work with NodeJS & MongoDB. Also I have a fair bit of knowledge about Linux system administration, including tools like Docker.

Projects

Most of the stuff I experiment with don't turn into real projects.

Some stuff has turned out well though, and that's what I have listed here.

  • Raqbot Raqbot is a NodeJS based Discord bot written in typescript. Its main feature is to keep track of statistics as of right now.
  • Raqclock Raqclock is a Raspberry pi based alarm clock, which runs in a kiosk-like web environment.
  • Mancave-Statistics Mancave-Statistics is a website which loads statistics from firebase & displays it in nice looking graphs & counters.
